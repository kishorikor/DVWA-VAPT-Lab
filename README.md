# Vulnerability Assessment and Penetration Testing on DVWA

## Objective
The objective of this project is to perform Vulnerability Assessment and Penetration Testing (VAPT) on Damn Vulnerable Web Application (DVWA) in a controlled lab environment to identify common web security vulnerabilities and understand their exploitation and mitigation.

## Target Application
Damn Vulnerable Web Application (DVWA)

## Lab Environment
- Target: DVWA
- Server: Apache (XAMPP)
- OS: Windows
- Browser: Chrome

## Tools Used
- Web Browser (Google Chrome)
- XAMPP (Apache Server)
- DVWA
- Command Prompt

## Disclaimer
This project was conducted in a controlled lab environment using DVWA for educational purposes only.
